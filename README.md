# Proyecto-Estadistica-EvolveAcademy
#### Víctor Horcas Puertas
Tercer proyecto realizado para el Master en Data Science de Evolve. Estadistica en Python, haciendo uso de Jupyter Notebook.

**Elección del dataset**: Extráido de *[Kaggle](https://www.kaggle.com/)*.
Mi Dataset -- > *[Students Social Media Addiction](https://www.kaggle.com/datasets/adilshamim8/social-media-addiction-vs-relationships)*.

El proyecto consiste en:
- Haciendo uso de mi dataset realizar:
    - Un análisis estadístico descriptivo partiendo de una pequeña exploración y análisis EDA hasta la detección de outliers y visualización de las distribuciones.
    - Inferencia y modelado del dataset: Correlación y regresiones lineal y logística.

- Haciendo uso de datos autogenerados:
    - Regresión lineal, usando las librerías de NumPy y Scikit - Learn.
    - Análisis de series temporales.
